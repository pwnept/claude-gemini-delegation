# Future Work

## Dave Code Audit

Preserve Dave as the exhaustive code-review workflow paired with the Gemini
Librarian role. Improve it before expanding its use: rank architectural and
high-impact call-chain findings first, trace every relevant function and link,
and impose a budget for low-impact edge cases so small bugs cannot consume the
audit before major risks are mapped.

## Project-local customization

Global managed installation is the default. Keep project-local installation as
an explicit option for repositories that later need genuinely different policy,
agents, or delegation behavior.
